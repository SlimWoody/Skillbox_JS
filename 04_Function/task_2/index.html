Задача 2

Создайте две функции для перевода градусов Цельсия в градусы Фаренгейта и, наоборот, градусов Фаренгейта в градусы Цельсия.

В первом случае функция принимает значение температуры в градусах Цельсия и возвращает его значение в градусах Фаренгейта. Формула для перевода:

F = C * 1.8 + 32

Во втором случае функция принимает значение температуры в градусах Фаренгейта и возвращает его значение в градусах Цельсия. Формула для перевода:

C = (F - 32) / 1.8

Вызовите каждую функцию с различными значениями и выведите результаты в консоль.

Пример 1: Вызов: celsiusToFahrenheit(25) Вывод: 77

Пример 2: Вызов: fahrenheitToCelsius(77) Вывод: 25
